# BIB OJT PJ
- **URL**
```
http://localhost:38000/



Note - If user left a text field empty , validation messages will show
  
```

- **Installation**
```
git clone https://github.com/AungKyawSoeOo/BIB_PJ.git
cd bib_project_starter-main_akso
npm install (to install node modules)
npm run build (to compile typescript files to js)
npm run dev (to run on localhost:8000 using nodemon)

open http://localhost:3000/ in the browser
```
